MacOS安装jupyter心路历程

发表于 2019-08-14 阅读次数：

用pip安装jupyter提示

我版本过低？？这能忍？
升级!

pip install --upgrade pip

出错了 让我加个–user 虽然我也不懂为啥要加

~pip install --user --upgrade pip
安装完成！运行！
~python -m IPython notebook

…… 为什么还是找不到阿喂！

永不服输的男人 吃过饭后继续战斗

运行python3，然后import要查看路径的包，再输入包的名称，即可查看安装路径 找到pip所在文件夹

参考这位老哥的经验
发现一个和pip重复的pip-19.1.1.dist-info文件夹，大概就是因为更新错地方然后出现了死循环，删它！

然后重新~ pip install --user jupyter

然后！在许多黄黄的WARNING中成功了！！！普天同庆！！策马奔腾！！（希望以后不会有奇奇怪怪的小毛病）

（话说我还不懂怎么解决下面这个mac自带2.7python即将过期的问题…… 应该到快过期的时候会有大佬出解决方案之类的吧哈哈哈)

阿 对了 进入以后是按ctrl+c退出 ……

又是充实的一天呢（才怪）

好的正准备上传 我发现我又不会用hexo了

发布文章 是要先cd到hexo的目录

然后用 hexo n "新的博客"才是阿！ 我一直在家目录新建个什么玩意……

PS：安装不成功这个问题已经纠结了我很久了，今天给它写下来似乎感觉思路会比较清晰一些，中间还有很多奇奇怪怪的步骤不过感觉好像没有什么卵用就没写进来了。总算是搞清楚了，明天开始新的折腾

等等 我刚pip show pip了一下

为什么……还是19.1.1版本……

是我输了……

试了一下~ jupyter notebook命令 发现用不了

大概意思是2.7的python没有安装 然后用py3找了一下发现确实jupyter装在py3了

这就很灵性了…… 以后只能用 ~python -m IPython notebook进入了
