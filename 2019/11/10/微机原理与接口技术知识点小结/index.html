微机原理与接口技术知识点小结

发表于 2019-11-10 阅读次数：

零散知识点

  • 处理器定时工作的时间单位从小到大：
    时钟 > 总线 > 指令周期

  • IO端口编址方式：独立编址 统一编址（51）

  • 数据总线DB双向，地址总线AB单向，控制总线CB单向（单根）

  • 非压缩性BCD => 8位2进制
    如十进制31D表示为：0000 0011 0000 0001
    通常用于处理ASCLL码

  • 对程序计数器PC的操作 是自动(通过加1)进行的

  • DATA区（00H到7FH）数据访问速度最快

  • 00H 经过DEC后变为 FFH

  • 程序寄存器ROM 用MOVC （MOVC @A , @A+DPTR）； 数据寄存器RAM 用MOVX

算术运算类

指令格式 说明
1. ADD / ADDC 加法（带进位）
2. SUBB 减法
3. MUL 乘法
4. DIV 除法
5. INC 加一
6. DEC 减一
7. DA A 十进制调整 ，指令修正，用于修正BCD码十进制加法计算结果。

控制转移类

无条件转移

指令格式 说明
1. SJMP rel 相对转移（2字节指令）转移范围较小时采用
2. AJMP addrll 绝对转移
3. LJMP addr16 长转移（3字节指令）

条件转移

指令格式 说明
1. JZ / JNZ 判断A的值 JZ（Jump if ACC is zero）： A=0：转移
2. CJNE 判断两数大小 比较结果影响标志位C 若 (1) >= (2) 则 C=0
3. DJNZ (Rn) <- (Rn-1)减一不为0，跳转目标地址。若已减到0，结束循环
* LCALL 调用子程序
* RETI 子程序最后一条指令必须是RETI

位操作

指令格式 说明
1. CLR / SETB 清零 /置位
2. ANL /ORL 与/或操作 结果保存在Cy中
3. JC/JNC JB/JNB JBC 位转移指令 判断Cy或bit

中断

中断入口

中断源 入口地址
外部中断0（INT0） 0003H
定时器/计数器0（T0） 000BH
外部中断1（INT0） 0013H
定时器/计数器1（T0） 001BH
串行口中断（TX或RX） 0023H

定时器/计数器控制寄存器TCON（Timer Control）

* 7 6 5 4 3 2 1 0
位符号 TF1 TR1 TF0 TR0 IE1 IT1 IE0 IT0
英文注释 Timer1 Overflow Timer1 Run Timer0 Overflow Timer0 Run Interrupt External 1 edge flag Interrupt 1 Type control bit Interrupt External 0 edge flag Interrupt 0 Type control bit

IE（中断允许控制寄存器）

用于控制所有中断源的开放或禁止，以及每个中断源是否被允许。

* 7 6 5 4 3 2 1 0
位序号 D0 D1 D2 D3 D4 D5 D6 D7
说明 外部中断0 定时/计数0 外部中断1 定时/计数1 串行口中断 定时2溢出中断（8052用） 无效位 全局中断位
位符号（写程序时直接引用） EX0 ET0 EX1 ET1 ES – – EA
